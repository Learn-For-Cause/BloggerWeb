## Getting Started

```bash
git clone https://github.com/Learn-For-Cause/BloggerWeb.git
cd nextjs-typescript-mdx-blog

yarn install
# or
npm install

yarn dev
# or
npm run dev
```

Your new site will be up at http://localhost:3000/
